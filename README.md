# portuguese_website_client_canalfestas
I created this website for my client. He has a company in Brazil that specializes in parties and children's entertainment.
I created this website using just a text editor.
Key Features
Modern Hero Section:High-impact header with a personalized call-to-action ("Transforming dreams into reality").
About Us: Dedicated section for company storytelling and mission.
Services Grid: Clean layout using modern icons to showcase party activities and recreation packages.
Smart Contact Section: Integrated social media links with custom icons for WhatsApp and Instagram.
Fully Responsive: Optimized for a seamless experience on desktops, tablets, and smartphones.

## Tech Stack (Tools Used)
This project was built using a "Back to Basics" approach, utilizing only a text editor and core web technologies:

HTML5: Semantic structure for better SEO and accessibility.
CSS3: Custom styling featuring Flexbox, CSS Grid, and responsive Media Queries.
Google Fonts: Using the *Poppins* font family for a friendly and modern look.
Font Awesome (v6.0): Vector icons for social media and service descriptions.
WhatsApp API (WA.ME): Direct "Click-to-Chat" functionality for instant lead generation.

## Visual Identity
Primary Color (Blue):`#0056b3` - Represents reliability and safety.
Secondary Color (Yellow):`#ffcc00` - Represents fun, energy, and childhood joy.
